Installation Guide

Learn how to install and set up our platform on your system.

System Requirements

Before you begin the installation process, make sure your system meets the following requirements:

  • Operating System: Windows 10/11, macOS 10.15+, or Linux (Ubuntu 18.04+, CentOS 7+)
  • Processor: Dual-core 2GHz or higher
  • Memory: 4GB RAM minimum, 8GB recommended
  • Disk Space: 2GB available space
  • Internet Connection: Broadband connection for downloads and updates

Important

Administrator privileges are required for installation on Windows and macOS. On Linux, you'll need sudo access.

Installation Steps

Step 1: Download the Installer

Visit our download page and select the appropriate installer for your operating system.

Step 2: Run the Installer

Double-click the downloaded file to start the installation process. Follow the on-screen instructions to complete the installation.

Command Line 
# For Linux/macOS users, you can also install via command line:
curl -sSL https://example.com/install.sh | bash

# For Windows users with PowerShell:
Invoke-WebRequest -Uri "https://example.com/install.ps1" -OutFile "install.ps1"
.\install.ps1

Step 3: Verify Installation

To verify that the installation was successful, open a terminal or command prompt and run the following command:

Command Line 
platform-cli --version

You should see the version number of the installed platform.

Installation Options

Silent Installation

For automated deployments, you can use the silent installation option:

Command Line 
# Windows
setup.exe /S /D=C:\Program Files\OurPlatform

# macOS
installer -pkg OurPlatform.pkg -target /Applications

# Linux
./installer.run --silent

Custom Installation

If you need to customize the installation, you can use the following options:

  • --prefix=PATH - Install to a different location
  • --skip-dependencies - Skip installing dependencies
  • --no-auto-update - Disable automatic updates

Next Steps

After successfully installing our platform, you should proceed to the Configuration section to set up your environment and then check out the Beginner Guide to start building your first application.
