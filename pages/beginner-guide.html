Beginner's Guide

A step-by-step guide to building your first application with our platform.

Prerequisites

Before you begin, make sure you have:

  • Installed our platform
  • Configured your environment
  • Basic knowledge of programming concepts
  • A text editor or IDE of your choice

Recommended Tools

We recommend using Visual Studio Code with our official extension for the best development experience.

Creating Your First Project

Step 1: Initialize a New Project

Open your terminal or command prompt and run the following command:

Command Line 
platform-cli init my-first-app

This will create a new directory called my-first-app with the basic project structure.

Step 2: Navigate to Your Project

Change to your project directory:

Command Line 
cd my-first-app

Step 3: Explore the Project Structure

Your project should have the following structure:

Directory Structure 
my-first-app/
├── src/
│   ├── controllers/
│   │   └── index.js
│   ├── models/
│   │   └── index.js
│   ├── views/
│   │   └── index.html
│   └── app.js
├── config/
│   └── default.yaml
├── tests/
│   └── app.test.js
├── package.json
└── README.md

Building a Simple Application

Step 1: Define Your Model

Create a new file src/models/user.js with the following content:

JavaScript 
// src/models/user.js
const { Model } = require('@platform/core');

class User extends Model {
  static schema = {
    name: { type: 'string', required: true },
    email: { type: 'string', required: true, unique: true },
    age: { type: 'number', min: 0 },
    createdAt: { type: 'date', default: Date.now }
  };
  
  getFullName() {
    return this.name;
  }
  
  isAdult() {
    return this.age >= 18;
  }
}

module.exports = User;

Step 2: Create a Controller

Create a new file src/controllers/user.js:

JavaScript 
// src/controllers/user.js
const { Controller } = require('@platform/core');
const User = require('../models/user');

class UserController extends Controller {
  async getUsers(req, res) {
    try {
      const users = await User.find();
      res.json(users);
    } catch (error) {
      res.status(500).json({ error: error.message });
    }
  }
  
  async createUser(req, res) {
    try {
      const { name, email, age } = req.body;
      const user = new User({ name, email, age });
      await user.save();
      res.status(201).json(user);
    } catch (error) {
      res.status(400).json({ error: error.message });
    }
  }
  
  async getUserById(req, res) {
    try {
      const user = await User.findById(req.params.id);
      if (!user) {
        return res.status(404).json({ error: 'User not found' });
      }
      res.json(user);
    } catch (error) {
      res.status(500).json({ error: error.message });
    }
  }
}

module.exports = new UserController();

Step 3: Update the Main Application

Modify src/app.js to include your new controller:

JavaScript 
// src/app.js
const { Application } = require('@platform/core');
const userController = require('./controllers/user');

const app = new Application();

// Register routes
app.get('/users', userController.getUsers);
app.post('/users', userController.createUser);
app.get('/users/:id', userController.getUserById);

// Start the application
app.start().then(() => {
  console.log('Application started successfully!');
}).catch(error => {
  console.error('Failed to start application:', error);
});

Running Your Application

To run your application, use the following command:

Command Line 
platform-cli start

Your application should now be running at http://localhost:3000.

Congratulations!

You've successfully created your first application with our platform. You can now test your API endpoints using tools like Postman or curl.
