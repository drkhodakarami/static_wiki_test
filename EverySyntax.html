Inline Code

Our platform uses several configuration files to control its behavior. The main configuration file is config.yaml, which is located in the following directory:

  • Windows: C:\Users\<username>\AppData\Local\OurPlatform\config.yaml
  • macOS: /Users/<username>/Library/Application Support/OurPlatform/config.yaml
  • Linux: /home/<username>/.config/ourplatform/config.yaml
